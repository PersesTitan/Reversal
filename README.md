## reversal(배열 또는 List)
설명 : reversal안에 배열 또는 List를 넣으면 순서가 반전됩니다.</br>
리턴 타입 : void</br>
지원 하는 배열 : Byte, Short, Integer, Long, Float, Double, Character, Boolean, byte, short, int, long, float, double, char, boolean, String, List</br>
예제 </br>
```java
코드
int[] i = {1, 2, 3};
reversal(i);
System.out.println(i);

출력
[3, 2, 1]

```
